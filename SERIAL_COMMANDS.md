# 小车串口命令说明

## 1. 串口参数

- 串口：UART0（PA10 TX、PA11 RX）
- 波特率：115200
- 数据格式：8N1，无流控
- LaunchPad：J21、J22 置于 1-2，连接 XDS Backchannel UART
- 每条命令发送后必须带回车或换行
- 命令字母不区分大小写

## 2. 参数说明

```text
方向 [速度RPM] [内侧轮速度百分比]
```

- 速度范围：0～1000 RPM
- 默认速度：200 RPM
- 省略速度时沿用上一次速度
- 内侧轮百分比范围：0～100，仅用于弧线转向
- 默认内侧轮百分比：50%
- 省略内侧轮百分比时沿用上一次设置

内侧轮百分比越接近 100，转弯半径越大；越接近 0，转弯越急。

## 3. 直线与原地转向

| 命令 | 功能 | 示例 |
|---|---|---|
| `F [速度]` | 前进 | `F 200` |
| `B [速度]` | 后退 | `B 200` |
| `L [速度]` | 原地左转 | `L 150` |
| `R [速度]` | 原地右转 | `R 150` |
| `X` | 主动制动停车 | `X` |

例如先发送 `F 300`，之后只发送 `F`，仍会以 300 RPM 前进。

## 4. 原地定角闭环转向

```text
TL 角度 [最大RPM]
TR 角度 [最大RPM]
```

| 命令 | 功能 | 示例 |
|---|---|---|
| `TL` | 原地向左转指定相对角度 | `TL 50.2` |
| `TR` | 原地向右转指定相对角度 | `TR 40.1` |

- 角度范围：0.1～360.0°，输入保留一位小数。
- 最大转速可省略，默认 180 RPM；例如 `TL 90.0 140`。
- 每次 `TL`/`TR` 开始时会自动把当前车头方向作为相对 0°，无需先发送 `Y0`。
- 控制器根据角度误差减速，并利用当前 Z 轴角速度提前制动。
- 接近目标后进入制动保持，并用滞回防止在容差边界反复启停抖动。
- 完成后输出 `ANGLE_TURN_DONE z_angle=... deg`。
- `X` 可随时取消定角转向并停车；发送任何普通运动命令也会取消定角控制。
- 定角转向期间禁止执行 `Y0`，串口会返回 `ANGLE_TURN_BUSY`。

首次测试前先手动低速左转并发送 `Y`。左转时角度应增加；如果角度减小，应将
`main.c` 中的 `ANGLE_TURN_LEFT_YAW_SIGN` 从 `1` 改为 `-1`，否则闭环方向错误。

## 5. 行进中弧线转向

| 命令 | 功能 | 示例 |
|---|---|---|
| `FL [速度] [百分比]` | 前进并左转 | `FL 200 70` |
| `FR [速度] [百分比]` | 前进并右转 | `FR 200 70` |
| `BL [速度] [百分比]` | 后退并向左转 | `BL 150 40` |
| `BR [速度] [百分比]` | 后退并向右转 | `BR 150 40` |

转弯强度参考：

- `80～100`：缓弯
- `50～79`：普通转弯
- `20～49`：急转弯
- `0～19`：内侧轮接近停止，转弯最急

示例：

```text
FL 300 80
FR 300 40
BL 200 20
X
```

## 6. 车轮编号

```text
前方

C（左前）    B（右前）
D（左后）    A（右后）
```

## 7. 无效命令

格式错误、超出参数范围或无法识别的命令会被忽略，不会改变当前运动状态。

旧的“四个RPM数值以空格分隔”协议已被遥控命令取代，当前不再支持。

## 8. 状态输出

电机状态串口输出当前已暂停：

```c
#define ENABLE_MOTOR_STATUS_UART (0)
```

需要恢复四个电机的目标转速、实际转速、编码器计数和 PWM 输出时，将其改为：

```c
#define ENABLE_MOTOR_STATUS_UART (1)
```

## 9. MPU6050 Z 轴相对转角

上电后必须让小车完全静止约 5 秒，等待串口出现：

```text
MPU6050 ready, WHO_AM_I=0x70. Send Y to read angle, Y0 to reset.
```

| 命令 | 功能 |
|---|---|
| `Y` | 返回当前 Z 轴相对转角，例如 `z_angle=32.5 deg` |
| `Y0` | 把当前位置设为 0° |

开始一次转角测量前，先将车头对准起始方向并发送 `Y0`，转弯过程中或结束后发送
`Y` 查看角度。模块水平安装且 Z 轴朝上时，按右手定则，逆时针通常为正、顺时针为负；
若模块安装方向相反，符号也会相反。

MPU6050 没有磁力计，当前结果由 Z 轴角速度积分得到，是相对角度，静止时也会缓慢漂移；
它不能作为长期无漂移的绝对航向。
