# 小车串口命令说明

## 1. 串口参数

- 串口：UART0（PA10 TX、PA11 RX）
- 波特率：115200
- 数据格式：8N1，无流控
- LaunchPad：J21、J22 置于 1-2，连接 XDS Backchannel UART
- 每条命令发送后必须带回车或换行
- 命令字母不区分大小写
- 上电后串口会打印欢迎信息与快速入门；随时发送 **`H`** 或 **`?`** 可查看完整命令列表
- 未知命令或格式错误会提示用法，并提示 `Send H for help`

## 2. 参数说明

```text
方向 [速度RPM] [内侧轮速度百分比]
```

- 速度范围：0～1000 RPM
- 默认速度：200 RPM
- 省略速度时沿用上一次速度
- 内侧轮百分比范围：0～100，仅用于弧线转向
- 默认内侧轮百分比：50%
- 省略内侧轮百分比时沿用上一次设置

内侧轮百分比越接近 100，转弯半径越大；越接近 0，转弯越急。

## 3. 直线与原地转向

| 命令 | 功能 | 示例 |
|---|---|---|
| `F [速度]` | 前进 | `F 200` |
| `B [速度]` | 后退 | `B 200` |
| `L [速度]` | 原地左转 | `L 150` |
| `R [速度]` | 原地右转 | `R 150` |
| `X` | 主动制动停车 | `X` |

例如先发送 `F 300`，之后只发送 `F`，仍会以 300 RPM 前进。

## 4. 原地定角闭环转向

```text
TL 角度 [最大RPM]
TR 角度 [最大RPM]
```

| 命令 | 功能 | 示例 |
|---|---|---|
| `TL` | 原地向左转指定相对角度 | `TL 50` |
| `TR` | 原地向右转指定相对角度 | `TR 40` |

- 角度范围：1～360°，整数即可。
- 转速可省略，默认 **100 RPM**（建议先用低速）；例如 `TL 90 80`。
- 每次 `TL`/`TR` 开始时会自动把当前车头方向作为相对 0°。
- 控制策略：定速旋转 → 按角速度提前滑行 → 若仍差一点则低速单向补转 → 结束。
- **不会反向补角**；最终误差大约 ±1°。
- 到位后使用滑行停车（不用反向主动刹车），减少车身抖动。
- 完成后输出 `ANGLE_TURN_DONE z_angle=... deg`。
- 若方向装反，会输出 `ANGLE_TURN_FAULT`，请改 `ANGLE_TURN_LEFT_YAW_SIGN`。
- `X` 可随时紧急制动取消；发送普通运动命令也会取消定角控制。
- 定角转向期间禁止执行 `Y0`，串口会返回 `ANGLE_TURN_BUSY`。

首次测试前先手动低速左转并发送 `Y`。左转时角度应增加；如果角度减小，应将
`main.c` 中的 `ANGLE_TURN_LEFT_YAW_SIGN` 从 `1` 改为 `-1`，否则闭环方向错误。

## 5. 行进中弧线转向

| 命令 | 功能 | 示例 |
|---|---|---|
| `FL [速度] [百分比]` | 前进并左转 | `FL 200 70` |
| `FR [速度] [百分比]` | 前进并右转 | `FR 200 70` |
| `BL [速度] [百分比]` | 后退并向左转 | `BL 150 40` |
| `BR [速度] [百分比]` | 后退并向右转 | `BR 150 40` |

转弯强度参考：

- `80～100`：缓弯
- `50～79`：普通转弯
- `20～49`：急转弯
- `0～19`：内侧轮接近停止，转弯最急

示例：

```text
FL 300 80
FR 300 40
BL 200 20
X
```

## 6. 车轮编号

```text
前方

C（左前）    B（右前）
D（左后）    A（右后）
```

## 7. 单电机控制

单独设置某一个电机的目标转速，不影响其他三个电机当前目标。

```text
A|B|C|D [转速RPM]
```

| 命令 | 功能 | 示例 |
|---|---|---|
| `A [转速]` | 单独控制电机 A（右后） | `A 200` |
| `B [转速]` | 单独控制电机 B（右前） | `B -150` |
| `C [转速]` | 单独控制电机 C（左前） | `C 180` |
| `D [转速]` | 单独控制电机 D（左后） | `D 0` |

- 转速范围：-1000～1000 RPM
- 省略转速时等价于 `0`（停该电机）
- **正转速 = 小车前进方向**（已按左右镜像符号换算到电机）
- 负转速 = 小车后退方向
- 会取消正在进行的定角转向；不会自动改其他轮目标
- 成功后回显，例如：`MOTOR_A chassis_rpm=200 motor_rpm=-200`

调试单轮时可配合 `M1` 看实测转速；全部停车仍用 `X`。

## 8. 无效命令

格式错误、超出参数范围或无法识别的命令会被忽略，不会改变当前运动状态。

旧的“四个RPM数值以空格分隔”协议已被遥控命令取代，当前不再支持。

## 9. 电机状态输出

输出四个电机的目标转速、实测转速、100 ms 编码器计数和 PWM 占空比。

| 命令 | 功能 |
|---|---|
| `M` | 等待下一组完整采样后打印一次（约 100 ms） |
| `M1` | 连续打印（约每 100 ms 一行） |
| `M0` | 停止连续打印 |

发送 `M` 后会先返回 `MOTOR_STATUS_WAIT`，随后输出一行：

```text
A[target_rpm=200,speed_rpm=198.0,counts_100ms=...,pwm_percent=...],B[...],C[...],D[...]
```

车轮对应：A 右后，B 右前，C 左前，D 左后。

## 10. 八路灰度读取（蓝光模块）

引脚：`AD0=PB11`，`AD1=PB5`，`AD2=PA1`，`OUT=PA14`（SysConfig 已用 `pin.$assign` 锁定）。
读取算法对齐官方 `Grayscale_Read`：选通 AD0/AD1/AD2 → 等待 50 µs → 读 OUT；并在电机干扰环境下对 OUT 做上拉+施密特与 3 次多数表决。
**请断开 J20**（PA1 板载上拉），否则 AD2 电平可能被拉偏，通道会错乱。
模块需 **5V** 供电；探头离地约 18 mm。灯亮应对应该路为 `1`。

| 命令 | 功能 |
|---|---|
| `G` | 读取一次 8 路数字量 |
| `G1` | 每 100 ms 连续打印 |
| `G0` | 停止连续打印 |

示例输出：

```text
GRAY X1:0 X2:0 X3:1 X4:1 X5:1 X6:0 X7:0 X8:0
```

蓝光模块建议离地约 18 mm。白底下探头对准目标颜色、板上灯亮时，对应通道一般为 `1`；灯灭一般为 `0`。

## 11. MPU6050 Z 轴相对转角

上电后必须让小车完全静止约 5 秒，等待串口出现：

```text
MPU6050 ready, WHO_AM_I=0x70. Send Y to read angle, Y0 to reset.
```

| 命令 | 功能 |
|---|---|
| `Y` | 返回当前 Z 轴相对转角，例如 `z_angle=32.5 deg` |
| `Y0` | 把当前位置设为 0° |

开始一次转角测量前，先将车头对准起始方向并发送 `Y0`，转弯过程中或结束后发送
`Y` 查看角度。模块水平安装且 Z 轴朝上时，按右手定则，逆时针通常为正、顺时针为负；
若模块安装方向相反，符号也会相反。

MPU6050 没有磁力计，当前结果由 Z 轴角速度积分得到，是相对角度，静止时也会缓慢漂移；
它不能作为长期无漂移的绝对航向。
